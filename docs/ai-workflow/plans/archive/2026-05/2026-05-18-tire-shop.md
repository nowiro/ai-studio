# Plan: tire-shop e-commerce demo

## Goal

Showcase a production-shaped e-commerce front-end in Angular 21 / Material 3:
faceted product browsing modelled on a generic tire-retail catalogue,
product detail with size matrix, cart with localStorage persistence,
multi-step checkout (frontend only, no payments). Stresses
**search/filter UX** and **state management without a global store** —
pure signals + a couple of services.

## Scope

| In                                                                                          | Out                                                        |
| ------------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| Catalogue with 50–100 mocked tire SKUs in a TypeScript-generated dataset                    | Real backend / inventory API                               |
| Facet filters: brand, size (width/profile/diameter), season, EU label, price, speed, load   | Real payments (Stripe / PayU / Przelewy24)                 |
| Sort: price asc/desc, popularity, EU label score                                            | User accounts / login (guest checkout only)                |
| Product detail page: gallery, specs table, reviews (mocked), "fits my car" tab              | Server-side rendering / Angular Universal                  |
| Cart drawer + cart page with quantity adjust, remove, totals                                | Internationalisation (Polish strings inline; English code) |
| 4-step checkout: contact → delivery → invoice → summary (Reactive Forms, validated jointly) | Email confirmations                                        |
| LocalStorage persistence for cart + last viewed                                             | Admin panel                                                |
| Skeleton loaders, empty-states, retry on filter combos with 0 hits                          | Mobile-app wrapper / PWA install banner v1                 |
| Vitest unit tests on filtering, sorting, cart math (≥80 % thresholds)                       | A/B test framework                                         |
| Playwright E2E: filter → add to cart → checkout → summary                                   |                                                            |

## Inputs

- `apps/personal-data-wizard/**` — Reactive Forms + Material stepper pattern

- `libs/shared-language/**` — toggle pattern if PL/EN later
