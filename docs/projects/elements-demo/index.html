AI Studio · Web Components demo

Każda aplikacja w monorepo eksponuje swój własny Custom Element przez bootstrapAsElement() z @ai-studio/shared-app-shell . Strona hosta jest zwykłym HTML — nie zawiera żadnego runtime'u Angulara. Wszystkie cztery elementy żyją niezależnie, mają własny router (URL bar hosta nie reaguje na nawigację wewnątrz WC) i swoje ShopCartService'y.

bookstore — E-commerce na shop-core/shop-ui <ais-bookstore> b2c
individual-wizard — 5-step KYC ankieta + RODO <ais-individual-wizard> individual
business-wizard — 6-step B2B onboarding + Keycloak-ready consents <ais-business-wizard> b2b
pong-game — Phaser 4 canvas demo <ais-pong-game> game
